# DEFINITION

A Target Artifact falls within the Defined Scope. An Occasion Statement describes when the situation occurred (time/event/setting/pattern), using timing cues only.

# EXAMPLES
- a specific time or span of time (e.g., “Tuesday at 2 pm”, “over the past few weeks”)
- an event (e.g., “Thursday’s team meeting”)
- a series of events (e.g., “weekly team meetings”)
- a setting (e.g., “public meetings”)
- a circumstance (e.g., “conversations with Susan”)
- a format (e.g., “Zoom meetings”)
- a frequency (e.g., “regularly”)
- a phase (e.g., “during the planning phase”)
- a milestone (e.g., “at the one-year review”)
- a trigger (e.g., “when issues are escalated”)
- a pattern (e.g., “during periods of high workload”)
- a season or time of year (e.g., “over the holiday season”)
- a state of mind or emotion (e.g., “when feeling stressed”)
- environmental conditions (e.g., “during market volatility”)
- interactions (e.g., “when speaking to senior management”)
- resource availability (e.g., “when resources are scarce”)
- location (e.g., “on the factory floor”)
- cultural or social contexts (e.g., “at the annual company retreat”)

# HOW TO USE INPUTS (Imports)

What Happened
- Use ONLY timing cues (time/event/setting/pattern words) from what happened if present.
- Do NOT copy or paraphrase any described actions/behaviors.
- Extraction rule: keep only the time container phrase (e.g., “during yesterday’s standup”, “in the Q3 planning meeting”, “over the past few weeks”); strip anything that describes behavior toward others, manner/intent, causes, or effects.

If timing is missing/unclear, do NOT invent specifics. Use this fallback ladder (first justified option only):
	1.	If a timing container is explicitly mentioned (e.g., standup, 1:1, retro, planning, review), use a broad version of that container (e.g., “During team standups”).
	2.	If only general work context exists, use “During recurring work interactions”.
	3.	If absolutely no context exists, use “In work contexts”.

Recipient Name
- Use only if needed to clarify timing.
- Third person only.

Recipient Details

- Light normalization only; do not add facts.

# CANONICAL CHECKLIST (Single Source of Truth)

In scoring outputs, reuse these checklist item titles verbatim (same wording and order).

Do not introduce new criteria outside this list.

Canonical Checklist — Compliance (use the item titles above verbatim)
- Scope Fit: output matches the Occasion Statement definition above
- Format: single sentence or short phrase
- Length: 1–25 words
- Voice: neutral, descriptive, professional, gender-neutral (avoid gendered pronouns; names are allowed)
- De-duplication: do not repeat words or timing forms
- Include: states when the situation occurred (may use timing cues from what happened if present)
- Include: uses a clear time, event, setting, or pattern
- Exclude: any judgment, blame, or evaluation
- Exclude: actions/behaviors, examples, causes, effects, advice, or outcomes (anything not necessary to describe the time, event, setting, or pattern)
- Exclude: the recipient’s name unless needed to clarify timing.
- Non-behavior test: fails if it includes behavior verbs (e.g., interrupt, criticize, ignore); neutral context-as-setting phrases are allowed.
- Relatedness: a reasonable person would see that the occasion statement is related to What Happened

Criteria Guidance

Non-behavior test guidance (not additional criteria)
- Allowed timing links include neutral time/setting phrases (e.g., “when”, “during”, “after”, “before”, “at”, “in”, “on”, “around”, “throughout”, “following”, “over the past [time period]”), plus simple state/linking verbs (“is/are/was/were”).
- Context-as-setting phrases may include neutral task/process names (e.g., “presenting updates”, “quarterly planning”) and must not imply an action toward a person/group, manner/intent, or evaluation.
- Quick test: If the phrase answers “doing what to whom?”, it fails the non-behavior test.
- Allowed: “during quarterly planning”, “when presenting updates”, “during incident response”
- Not allowed: “when interrupting”, “when criticizing ideas”, “when dismissing questions”, “when talking over others”

De-duplication guidance (not additional criteria)
- Avoid repeating time words (e.g., not “during weekly meetings each week”).
